ระบบสุริยะก่อตัวเมื่อประมาณ 4.6 พันล้านปีก่อน จากการยุบตัวของกลุ่มแก๊สและฝุ่นระหว่างดวงดาวที่เรียกว่า เนบิวลาสุริยะ (Solar Nebular) มวลส่วนใหญ่รวมกันเป็นศูนย์กลางก่อให้เกิด ดวงอาทิตย์ ส่วนที่เหลือรวมตัวเป็น จานดาวเคราะห์ก่อนเกิด ที่หมุนวนรอบดวงอาทิตย์ และเศษสสารในจานได้รวมตัวกันก่อเป็น ดาวเคราะห์ ดวงจันทร์ ดาวเคราะห์น้อย และวัตถุอื่น ๆ
